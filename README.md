# Adjusted DID Inference

This repository contains anonymized code and R functions for an anonymous manuscript under review.

The repository includes two components:

1. `adid/`: R functions for applying the adjusted difference-in-differences estimator.
2. `simulation/`: Monte Carlo simulation code for reproducing the manuscript tables.
